== Description ==
[PhpSword Favicon Manager WordPress plugin](http://69plugins.com/free-plugins/phpsword-favicon-manager/ "PhpSword Favicon Manager WordPress plugin Link at 69plugins.com") lets you add a favicon to your WordPress website. Favicon is a small image or logo displayed in the address bar of a web browser. By using this plugin codes, you can easily upload and change favicon icon on your website built in WordPress.


= PhpSword Favicon Manager WordPress Plugin Features =
* Very simple and easy to use.
* No complex menus or setting.
* Just upload an image and change your website's default favicon logo.
* Enable or disable the favicon code on the website.
* Store two favicon logos and choose any one of them to display on the website.
* Compatible with almost every version of WordPress.

== Installation ==
= Search and Install from WordPress admin panel =
1. If you are logged in inside your WordPress admin panel, go to Plugins menu and click "Add New" button.
2. On Install Plugins page, under Search navigation menu, enter "PhpSword Favicon Manager" in the search bar and click search plugins.
3. You will find PhpSword Favicon Manager plugin By Pradnyankur Nikam in the list.
4. Click "Install Now" button to install the plugin or click "Details" to read more. Install now link is also available on details popup window.
5. Once installed you need to activate the plugin to use it on your website.

= Manual Installation =
1. Click "Download Version 1.0" button on plugin info page to download and save the plugin on your computer.
2. You will get "phpsword-favicon-manager.zip" file.
3. Login to your WordPress admin panel, go to Plugins -> Add New -> select "Upload" navigation on the page. Click the browse button, select downloaded "phpsword-favicon-manager.zip" file from your computer and click Install Now button. Go to plugins and activate the PhpSword Favicon Manager plugin from the list. OR Extract "phpsword-favicon-manager.zip" file and upload all the files inside "phpsword-favicon-manager" folder to your website's "wp-content/plugins" directory. Now login to your WordPress admin panel go to plugins and activate the PhpSword Favicon Manager plugin from the list.
4. Reload WordPress Admin Panel & you will get a new menu "PhpSw Favicon" on left sidebar.
5. Click and open "PhpSw Favicon" page. Here you can upload your own favicon images.
6. You can also enable or disable the favicon code by selecting respected dropdown menu.

== Frequently Asked Questions ==
= What is Favicon in WordPress or other website? =
A favicon (short for "favorites icon") is an icon associated with a website or webpage intended to be used when you bookmark the web page. Web browsers use them in the URL bar, on tabs, and elsewhere to help identify a website visually.

= What are standard image resolutions for a favicon? =
For IE, Microsoft recommends 16x16, 32x32 and 48x48 favicon.ico file, for iOS, Apple recommends resolutions at most 180x180 while some other platforms look for PNG files with various resolutions, like the 96x96 pixels.
